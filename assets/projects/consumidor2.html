2015 | Brazil's Consumer Protection Agency
Mobile design / Prototype / UX
The challenge was to create a mobile version (IOS) for Brazil's Consumer Protection portal - www.consumidor.gov.br
I was responsible for helping them to prioritize and distribute the content, information architecture and for create an "app like" layout based on the current visual identity.
I create 34 static screens using Illustrator and 8 navigable screens in Axure 8.

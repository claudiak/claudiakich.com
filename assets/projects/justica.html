2014 | Ministério da Justiça
Web Design / Bootstrap 3 / Mobile / Acessibilidade / E-Gov
Fui contratada como Web Designer para criar o novo layout do portal do Ministério da Justiça. O desafio era implementar um novo layout respeitando as melhores práticas de UI / UX e acessibilidade aproveitando o gerenciador de conteúdo atualmente utilizado pela casa – Lumis Portal.

Optei por implementar o framework Bootstrap 3.0 e encontrei alguns desafios para integrá-lo com o Lumis, via XML.

Após levantar os requisitos com a Casa e aprovar a nova Arquitetura da Informação, implementei e testei o Bootstrap e treinei a equipe de desenvolvedores para efetuarem a manutenção, através de workshops e video tutoriais.
